# DotfilesGtk
A GTK 2/3+ theme for my [dotfiles](https://github.com/bossley9/dotfiles)

## Table of Contents
1. [Installation](#installation)

## Installation <a name="installation"></a>
1. Download all dependencies:
    ```
    yarn
    ```
    or
    ```
    npm install
    ```
2. Build the theme:
    ```
    yarn build
    ```
    or
    ```
    npm run build
    ```
